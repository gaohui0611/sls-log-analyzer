# 🧪 快速测试指南

## ✅ 新功能已实现

### 功能1：自定义分析提示词
用户可以输入自定义的分析要求，指导AI关注特定方向。

### 功能2：日志详情展示
在报告中展示AI分析所依据的完整日志详情（前10条）。

---

## 🚀 快速开始

### 1. 启动服务器
服务器已启动在：**http://localhost:3002**

### 2. 访问系统
打开浏览器访问：http://localhost:3002

### 3. 测试步骤

#### 步骤1：配置认证（如果还没配置）
1. 进入"系统设置"页面
2. 使用"cURL手动同步"方式配置认证

#### 步骤2：配置项目（如果还没配置）
1. 进入"项目管理"页面
2. 添加一个测试项目

#### 步骤3：配置AI（如果还没配置）
1. 进入"AI配置"页面
2. 配置AI提供商和API Key

#### 步骤4：测试新功能
1. 进入"日志分析"页面
2. 选择项目和时间范围
3. **在"自定义分析提示词"输入框输入**：
   ```
   请重点分析错误日志的类型和发生频率，提供具体的解决建议
   ```
4. 点击"开始分析"
5. 等待分析完成
6. 查看结果：
   - 检查AI分析是否符合自定义要求
   - 滚动到底部查看"📋 分析依据日志详情"
