认证状态

⏳ 检查中...

点击"一键自动同步"会全自动完成：读取 Chrome 登录缓存（含 HttpOnly Cookie）→ 自动抓取 CSRF Token → 验证。
无需新开窗口、无需重新登录、无需书签。
首次使用：会弹一次系统对话框询问"是否允许访问钥匙串"，点「始终允许」（之后不再弹）。
前提：用 Chrome 登录过阿里云 SLS 控制台。

书签同步（备选）

通常无需使用此方式——「一键自动同步」已能自动抓取 CSRF Token。
仅当一键同步失败时，可用书签在 SLS 页面手动抓取 Token 作为补充。

  1. 拖拽下方按钮到浏览器书签栏
  2. 访问 阿里云 SLS 控制台
  3. 确保已登录后，点击书签栏上的"🔖 SLS同步"
  4. 看到"同步成功"提示即完成（cookies 自动读取 + token 抓取）
🔖 SLS同步

⚠️ 重要：必须在 sls.console.aliyun.com 页面点击书签才有效。
若提示 token 已过期，刷新 SLS 页面后重新点击即可。Cookies 续期请用「一键同步」。

备用方式：cURL

如果自动同步不工作，可以使用 cURL 方式：

  1. 打开浏览器开发者工具 (F12)
  2. 访问 阿里云 SLS 控制台
  3. 执行一次日志搜索操作
  4. 在 Network 面板找到 getLogs.json 请求
  5. 右键 → Copy → Copy as cURL
  6. 将 cURL 命令粘贴到下方
